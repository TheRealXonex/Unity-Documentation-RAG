# Unity-Documentation-RAG
This is a unity RAG for the Unity Docs

It seperates prompts into 3 seperate sections:
1. Overview - a brief overview of the item asked for in the Unity Docs using bullet points
2. Code Sample - provides sample code depending on if its relevant to the topic of the request otherwise will result in no code generated
3. The directory - gives where to find the full html page of information on the project for a more in depth look
